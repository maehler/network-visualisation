Documentation

General

The network is visualized as nodes (genes) connected by edges. The size of the nodes correspond to the degree of the nodes, i.e. how many other nodes they are connected to. Edges can either be directed (indicated by an arrow), or undirected.

Network modules

Network modules can be visualized separately by choosing the id of the module to visualise. These are generally quick to load, easy to manipulate and navigate. By leaving the input empty and clicking "Visualize module", the entire network will be drawn. This is an expensive operation that will take time, and the interface will likely become very slow. Only do this if you really have to.

Gene neighborhoods

Instead of visualizing a pre-defined network module, you can also visualize the neighborhood of a particular gene. Enter the identifier of the gene that you are interested in and click "Visualize neighborhood". This will display the first degree neighbors of the gene, i.e. the genes that are connected directly to the gene of interest.

Finding genes in the network

If there are a lot of nodes in the network, it can be difficult to visually find the gene of interest. Therefore you can search for a gene in the current network, and it will be highlighted if it exists. Simply enter a gene identifier and click "Select gene of interest".

Annotations

Nodes can be colored according to Gene Ontology (GO) or protein family (Pfam) terms. Enter an annotation term ID and a the color that should be associated with it and click "Color genes of interest". The colors can be reset by clicking "Reset node colors".

One current limitation is that a node can only have one color at a time. If several terms are colored, and a gene belongs to more than one of them, the node will have the last color applied.

Hiding edges

Edges can be hidden based on whether they are directed or not. Simply check "Hide directed edges" or "Hide undirected edges".

GO enrichment

Functional enrichment can be tested by clicking the button labelled "GO enrichment". This will test for enrichment of GO terms in the currently displayed network using gofer2. The test will run in the background, and once the results are ready a table with the results will appear.

Export

The currently displayed network can be exported in GraphML format. This can be opened in most graph software, such as Gephi or Cytoscape. One caveat is however that different software might render the file differently. In some cases it might not be able to have a mix of directed and undirected edges or node names might be missing.
